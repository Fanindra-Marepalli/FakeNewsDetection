# 📰 Fake News Detection

This project uses Natural Language Processing and Machine Learning to classify news articles as real or fake.

## 🚀 Features
- NLP with TF-IDF/CountVectorizer
- Classification using Logistic Regression / Naive Bayes
- Accuracy metrics and model evaluation

## 📂 Dataset
The dataset is too large to upload to GitHub.  
Download it from [this Kaggle link](https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset) and place it in a folder called `/data`.
